## CBA Configs für die Server
### Standard
- **Medical Übungsplatz**
  - ``!NACH DEN MEDICAL EINSTELLUNGEN LADEN!``
  - Einstellungen für die AI um sie als medizinische Testsubjekte verwenden zu können
- **Medical**
  - Medizinische Standard-Einstellungen

### Extra
- **Medical Ultralight**
  - Stark vereinfachte KAT Einstellungen für KAT-less gebaute Missionen. Benötigt keine Loadoutveränderungen.
  - Disabled Module für _Atemwege_, _Pneumothorax_, _SpO₂_, _IVs/IOs_ & _Tourniquet Debuffs_
  - Verringerte Bruch-Chance
  - Schnelleres TXA/EACA
  - ``!BEACHTEN!`` Saline/Medikamente (Ausnahme EACA) können ab EHB ohne Aufsicht gelegt werden!

_Einstellungen in das Textfeld des Addon Options "Import"-Button Copy-Pasten_
